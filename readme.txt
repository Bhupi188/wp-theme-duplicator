== Description ==

**Copy of any Themes from Dashboard**  Create copy of any themes from your admin dashboard without using FTP by just one click!

Nothing required for this plugin, you just need to install and activate this plugin and you can see "Copy" link on each theme box on Appearance page.
Now you just need to click on any of theme Copy button and that theme's copy will be automatically Created in theme directory.

= Download Plugin =

If you want to download any plugin from your wordpress admin panel's Plugins page, then use our other plugin - [Download Plugin](https://wordpress.org/plugins/wp-theme-duplicator)

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Extract the zipped file and upload the folder `download-theme` to `/wp-content/plugins/` directory.
